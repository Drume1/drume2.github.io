<?php
declare (strict_types = 1);

use think\Response;

class Check1
{
    /**
     * 处理请求
     *
     * @param \think\Request $request
     * @param \Closure       $next
     * @return Response
     */
    public function handle($request, \Closure $next)
    {
        //
        if ((Session::get("sb")==Session::get("token")&&!empty(Session::get("sb"))&&!empty(Session::get("token")))){

            return $next($request);
        }
        else{
            echo Session::get("sb");

这段PHP代码定义了一个位于app\iddleware命名空间下的中间件Check1。核心功能如下：
    实现了一个基础的Session令牌验证机制。
    在handle方法中，检查Session中"sb"和"token"两个值是否相等且都不为空。
    如果验证通过，则放行请求到下一个中间件/控制器。
    如果验证失败，会输出两个Session值，并返回错误响应"虽然我是新手,但是懂的一点token验证什么的"。
