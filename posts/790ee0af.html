传输方式为ASCII

FTP 有两种传输模式：

•ASCII 模式（默认）：会转换换行符（可能破坏文件）

•二进制模式（Binary）：原样传输，保留每个字节

这点一会要用到

#!/bin/bash

echo "[*] Resetting website from backup..."

# Remove current site
rm -rf /var/www/html/*
# Restore from backup
cp -r /opt/backup/* /var/www/html/
# Set correct ownership
chown -R www-data:www-data /var/www/html/
