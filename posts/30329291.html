helder–>root

没什么信息，传一个pspy64进来看一下定时任务

2026/03/29 14:11:01 CMD: UID=0     PID=765    | /bin/sh -c nc -vv -q 1 localhost 10000 > /root/.local/out && if [ "$(cat /root/.local/helder.txt)" = "$(cat /home/helder/passwd.txt)" ] ; then chmod +s "/usr/bin/$(cat /root/.local/out)" ; fi

通过构造让 helder.txt与 passwd.txt内容相同，并向本地 10000 端口写入 bash，一分钟后计划任务就会为 /usr/bin/bash设置 SUID 位，从而获得一个 root 权限的 shell
