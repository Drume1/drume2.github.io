问了ai说是FFmpeg 是一个音视频处理工具，主要用于处理多媒体文件

那么尝试将/root/root.txt文件转换成音频文件，然后进行读取

ai给出了下面的命令

sudo ffmpeg -f s16le -i /root/root.txt -c:a pcm_s16le /tmp/root_pcm.wav
sudo ffmpeg -f s16le -i /root/root.txt -c:a copy /tmp/root_raw.wav
sudo /usr/bin/ffmpeg -f s16le -i /root/root.txt -f wav /tmp/root.wav

这几个命令都可以将/root/root.txt文件转换成wav音频文件，然后再strings文件即可得到flag
